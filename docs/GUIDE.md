# DeskRPG User Guide

2D 픽셀아트 멀티플레이어 가상 오피스 게임.
캐릭터를 만들고, 채널(게임방)에 입장하여, AI NPC와 대화하고, 다른 플레이어와 실시간 소통하세요.

**Live:** https://opentown.dante-labs.com

---

## 1. 시작하기

### 1.1 회원가입 / 로그인

게임에 접속하면 로그인 화면이 나타납니다.

- **아이디 + 비밀번호**로 회원가입 또는 로그인
- 닉네임은 게임 내 표시명으로 사용됩니다

### 1.2 캐릭터 생성

로그인 후 캐릭터를 생성합니다.

- **프리셋 선택**: 20종의 직업별 프리셋 (개발자, 디자이너, 기획자, PM, 마케터, CEO 등)
- **커스텀 편집**: 체형(남/여), 피부색, 눈색, 헤어, 상/하의, 신발, 액세서리 등 자유 조합
- **랜덤**: 랜덤 버튼으로 무작위 외형 생성
- 캐릭터는 여러 개 만들 수 있으며, 캐릭터마다 다른 채널에 참여할 수 있습니다

### 1.3 채널 참여

캐릭터 선택 후 채널 목록에서 참여할 방을 선택합니다.

- **공개 채널**: 누구나 입장 가능
- **비공개 채널**: 비밀번호 입력 필요
- **초대 코드**: 친구가 공유한 초대 코드로 직접 참여
- **채널 만들기**: 직접 채널을 생성할 수도 있습니다

## 2. 게임 화면

채널에 입장하면 2D 픽셀아트 맵이 나타납니다.

### 2.1 이동

- **클릭 이동**: 맵의 원하는 위치를 클릭하면 캐릭터가 자동으로 이동
- **키보드 이동**: 방향키(↑↓←→) 또는 WASD로 이동
- 다른 플레이어와 NPC는 실시간으로 움직이는 모습이 보입니다

### 2.2 상단 헤더 바

게임 화면 상단에 다음 요소가 있습니다:

| 요소 | 설명 |
|------|------|
| **채널명 — 캐릭터명** | 현재 채널과 내 캐릭터 이름 |
| **접속자 수** | 현재 채널에 접속 중인 플레이어 수 |
| **회의실** | 회의실 모드로 전환 |
| **태스크** | NPC에게 할당된 태스크 보드 열기 |
| **메뉴 (캐릭터명 ▼)** | 통합 메뉴 드롭다운 |

### 2.3 통합 메뉴

캐릭터 이름 드롭다운을 클릭하면 통합 메뉴가 열립니다:

- **캐릭터 편집** — 캐릭터 외형 수정 페이지로 이동
- **NPC 고용** — AI NPC를 채널에 배치 (채널 소유자만)
- **채널 설정** — 채널 이름, 비밀번호, 공개 설정 변경 (채널 소유자만)
- **채널 초대** — 초대 링크/코드 공유
- **알림** — 게임 내 알림 확인 및 읽음 처리
- **언어** — 인터페이스 언어 변경 (한국어, English, 日本語, 中文)
- **채널 나가기** — 채널 목록으로 돌아가기
- **로그아웃** — 계정 로그아웃

## 3. 채팅

### 3.1 채널 채팅

맵 위에서 다른 플레이어 근처에 가면 채팅이 활성화됩니다.

- 왼쪽 채팅 패널에서 메시지를 입력
- 플레이어 근처에 있어야 채팅 입력이 가능합니다
- 메시지를 보내면 캐릭터 머리 위에 말풍선이 표시됩니다

### 3.2 NPC 대화

NPC 근처에서 클릭하면 1:1 대화가 시작됩니다.

- **대화 탭**: NPC와 자유롭게 대화
- **태스크 탭**: NPC가 할당한 태스크 목록 확인
- NPC는 AI로 구동되며, 설정된 페르소나에 따라 다른 성격으로 응답합니다
- 대화 중 NPC 응답이 스트리밍으로 표시됩니다

### 3.3 NPC 컨텍스트 메뉴

NPC를 우클릭(또는 길게 클릭)하면 컨텍스트 메뉴가 나타납니다:

- **호출** — NPC를 내 위치로 불러옴
- **복귀** — 호출한 NPC를 원래 자리로 돌려보냄
- **대화** — NPC와 대화 시작

## 4. NPC 관리 (채널 소유자)

채널 소유자는 AI NPC를 고용하고 관리할 수 있습니다.

### 4.1 NPC 고용

메뉴 → **NPC 고용**에서:

1. **이름** 입력
2. **AI 게이트웨이 연결** (OpenClaw 등) — 기존 에이전트 선택 또는 새로 생성
3. **페르소나 설정** — 프리셋(접수원, PM, 팀리더 등) 선택 또는 커스텀 작성
   - **정체성**: NPC의 역할, 성격, 말투
   - **영혼** (고급): KPI, 갈등 해결 규칙, 상세 보이스/톤
4. **외형 선택** — 프리셋 또는 커스텀 LPC 캐릭터
5. **맵에 배치** — 타일을 클릭하여 NPC 위치 지정

### 4.2 NPC 편집 / 해고

채팅 패널에서 NPC 대화 중 톱니바퀴(⚙) 아이콘:

- **편집** — NPC 이름, 페르소나, 외형 수정
- **해고** — NPC를 채널에서 제거
- **대화 초기화** — NPC와의 대화 기록 초기화

### 4.3 채널당 제한

- 최대 **10개 NPC** per 채널

## 5. 회의실

상단 **회의실** 버튼을 클릭하면 회의 모드로 전환됩니다.

### 5.1 회의 시작

1. **회의 주제** 입력
2. **참가자 선택** — 채널의 NPC 중 회의에 참여할 NPC 선택
3. **설정 조정**:
   - 최대 턴 수
   - 자동/수동 모드
   - NPC별 PASS 정책
4. **회의 시작** 클릭

### 5.2 회의 진행

- **자동 모드**: NPC들이 자동으로 발언 순서를 돌아가며 대화
- **수동 모드**: 일시정지 후 수동으로 다음 턴 진행
- 채팅 입력으로 직접 발언 가능 (플레이어도 회의 참여)
- 각 NPC는 의장(C 표시)이 될 수 있으며, PASS/발언 여부를 투표합니다

### 5.3 회의록

회의 종료 후:

- **자동 AI 요약** — 주요 주제, 결론 자동 생성
- **전체 대화 기록** 보기
- **내보내기** — 마크다운 파일 또는 클립보드 복사
- 과거 회의록은 회의실 화면에서 조회 가능

## 6. 태스크 관리

NPC와의 대화 중 NPC가 태스크를 생성할 수 있습니다.

### 6.1 태스크 보드

상단 **태스크** 버튼으로 칸반 보드를 열 수 있습니다:

- **대기중** — 아직 시작하지 않은 태스크
- **진행중** — 현재 작업 중인 태스크
- **완료** — 완료된 태스크

### 6.2 NPC별 필터

태스크 보드 상단에서 NPC 이름으로 필터링 가능합니다.

## 7. 채널 설정 (채널 소유자)

메뉴 → **채널 설정**에서:

| 설정 | 설명 |
|------|------|
| **채널 이름** | 채널 표시 이름 변경 |
| **설명** | 채널 소개 텍스트 |
| **공개/비공개** | 공개 설정 전환 |
| **비밀번호** | 비공개 채널 비밀번호 설정/변경 |
| **AI 게이트웨이** | OpenClaw 게이트웨이 URL 및 토큰 설정 |
| **멤버 관리** | 멤버 목록 확인, 강퇴 |

채널 생성 시 3가지 맵 템플릿 중 선택:

| 맵 | 설명 |
|----|------|
| **오피스** | 책상, 회의 공간, 로비 |

## 10. 키보드 단축키

| 키 | 동작 |
|----|------|
| `↑↓←→` 또는 `WASD` | 캐릭터 이동 |
| `/` | 가장 가까운 NPC에게 말걸기 |
| `Enter` | 채팅 입력 포커스 |
| `ESC` | 모달/대화창 닫기, NPC 배치 취소 |

## FAQ

**Q: NPC가 응답하지 않아요.**
A: AI 게이트웨이(OpenClaw)가 올바르게 설정되어 있는지 확인하세요. 채널 설정에서 게이트웨이 URL과 토큰을 확인합니다.

**Q: 채널이 보이지 않아요.**
A: 비공개 채널은 초대 코드가 있어야 참여할 수 있습니다. 채널 소유자에게 초대 코드를 요청하세요.

**Q: 캐릭터 외형을 변경하고 싶어요.**
A: 게임 내 메뉴 → **캐릭터 편집**에서 언제든 변경할 수 있습니다.

**Q: 회의 중 NPC가 PASS만 해요.**
A: NPC의 PASS 정책을 확인하세요. NPC 고용 시 또는 편집에서 회의 PASS 정책을 조정할 수 있습니다.
